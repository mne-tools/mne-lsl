neurodecode.decoder.BCIDecoder¶

class neurodecode.decoder.BCIDecoder(amp_name, classifier=None, buffer_size=1.0, fake=False, label=None)¶

Decoder class

The labels order of self.labels and self.label_names match likelihood orders computed by get_prob().

Parameters

  • amp_name (str) – Connect to a specific stream.

  • classifier (str) – The (absolute) path to the classifier file (.pkl)

  • buffer_size (float) – The signal buffer’s length in seconds

  • fake (bool) – If True, load a fake decoder

  • label (mp.Variable('i')) – Used for adaptive classifier, share the actual trial label
