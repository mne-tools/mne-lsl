Installation¶

Here we list the multiple options available to install NeuroDecode.

Notice: NeuroDecode has been tested with Python 3.7 only. We advise you to install Python thanks to miniconda or conda.

Dependencies¶

NeuroDecode requires the use of external libraries:

  • numpy

  • scipy

  • scikit learn

  • mne

  • pylsl

  • …

The full list of dependencies can be found in setup.py.

Requirements¶

During the installation, NeuroDecode requires to provide two folders paths for:

  • the protocols’ scripts and the subjects’ config files

  • the subjects’ data

Create those two folders before starting the installation.

Option 1. From source¶

Notice: This installation requires pip. It is included in the installation of Python, miniconda or conda. If not reconized, add its path to the environment path.

  1. Copy or clone NeuroDecode from the project’s GitHub page

  2. From the NeuroDecode folder, run:

    • For Windows:

    $ .\INSTALLER.cmd

    • For Linux:

    $ ./INSTALLER.sh

    You will be asked to provide the absolute paths of the two previously created folders.
